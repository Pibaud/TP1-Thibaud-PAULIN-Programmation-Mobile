# TP1 Thibaud PAULIN

Ce dépôt contient le code source de l'ensemble des exercices réalisés dans le cadre du TP1 de l'UE HAI811I - Programmation Mobile.
Le choix a été fait d'utiliser Kotlin comme langage au lieu de Java, afin de découvrir un nouveau langage qui semble être de plus en plus populaire dans le développement mobile.

Pour l'exercice de l'application des itinéraires de trains, la clé API a été mise dans le fichier local.properties par souci de sécurité et pour mettre en place de bonnes pratiques.

Enfin, la vidéo étant trop lourde pour être importée dans ce repository, vous la trouverez à la racine du zip déposé sur Moodle.
